F1 Manager 2025 - Prototype

Arquivos principais:
- index.html
- style.css
- data.js
- script.js
- imagens_f1_manager_2025.csv

Pastas de imagens (todas vazias, você deve adicionar as imagens):
- assets/logos -> logos das escuderias (ex: mclaren.png, ferrari.png...)
- assets/faces -> avatares dos pilotos (ex: max_verstappen.png...)
- assets/managers -> avatares dos gerentes (ex: manager_andrea_stella.png, manager_custom_01.png...)
- assets/flags -> bandeiras por código de país (ex: br.png, gb.png...)
- assets/tracks -> imagens de pistas (opcional, você pode ligar depois no layout)
- assets/ui -> fundos, capas e outros elementos visuais

Para jogar:
1. Suba esta pasta inteira para um repositório no GitHub.
2. Ative o GitHub Pages apontando para a branch com estes arquivos.
3. Abra o endereço do GitHub Pages no navegador (PC ou celular).
4. Adicione as imagens nas pastas corretas para deixar o visual completo.

A lógica de simulação está em script.js, e os dados da temporada em data.js.
